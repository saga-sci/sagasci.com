Saga Scientific

Saga Scientific develops artificial intelligence solutions for robotics, simulations, and embedded systems. These solutions apply to dual-use aerospace technologies. The company designs autonomous navigation algorithms, creates high-fidelity models in virtual environments for testing, and implements real-time control systems in hardware. This work supports unmanned vehicles, sensor fusion, and command and control architectures.

Expertise

Specializes in robotics with 20 years of experience. Expertise includes AI for robotics, algorithm optimization, management of multi-million-dollar projects, simulations and digital twins using the Robot Operating System (ROS), system integration for defense applications, embedded systems, sensor firmware, and computer-aided design (CAD).
